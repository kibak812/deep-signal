# 딥 시그널

`딥 시그널`은 침수된 데이터 심해를 내려가며 신호 카드로 덱을 만드는 웹 기반 싱글플레이 덱 빌딩 로그라이크입니다. 모든 콘텐츠, 세계관, 카드, 유물, 적, 이벤트, 화면 구성은 독창적으로 제작되었습니다.

## 실행

```bash
npm run dev
```

브라우저에서 `http://127.0.0.1:4173` 또는 `http://localhost:4173`을 엽니다.

## 검증

```bash
npm test
npm run build
npm run balance
npm run audit
```

## 구현 범위

- 시작 화면, 새 런, 저장된 런 미리보기와 덮어쓰기 확인이 포함된 이어하기, 설정, 게임 정보, 기록, 코덱스, 가이드 화면
- 시작 화면의 캐릭터 패널: `에코 다이버`의 주력 전략, 시작 체력, 에너지, 크레딧, 덱 수, 시작 유물, 시작 덱 구성 표시
- 게임 정보 화면의 조작 안내, 제작진, 크레딧, 이용 안내·라이선스
- 위험과 보상, 다음 분기, 적 후보, 보스 특징, 상점과 휴식에서 얻을 수 있는 자원을 설명하는 절차 생성 경로 맵. 각 막은 초반 성장, 후반 엘리트·상점 선택, 휴식, 보스전으로 이어집니다.
- 카드 71장, 보상 카드 63장, 유물 31종, 일반 적 15종, 엘리트 5종, 보스 3종, 이벤트 20종
- 뽑기/손패/버림/소멸 더미, 전투 중 더미 점검 모달, 에너지, 방어도, 상태 이상, 적 의도, 손패 카드별 사용 후 변화 미리보기, 이번 턴 추천 카드와 추천 이유 표시, 6장 이상 손패도 데스크톱에서 잘리지 않는 부채꼴 배치
- 카드 보상 선택 전 추천 카드/추천 유물/고른 뒤 덱 변화를 상단에 짧게 보여주는 보상 요약, 덱 크기/평균 비용/중복/종류 비율 변화 안내, 받지 않아도 좋은 상황 안내, 추천 보상 표시, 엘리트/보스/특수 전투 후 유물 3개 중 1개 선택과 추천 유물 및 현재 덱에 맞는 이유 안내, 카드 구매, 제거, 강화, 회복, 유물 구매, 상점 구매 뒤 잔액/덱 맞춤도/정비 결과 안내, 상점과 휴식 버튼의 추천 표시, 이벤트 선택지의 위험/보상과 추천 선택 표시, 이벤트/마켓/세이프룸의 현재 위치와 추천 판단 요약, 강화/제거 대상 선택 전 체력·덱 크기·비용을 먼저 보여주는 카드 선택 브리핑, 선택 직후 체력·크레딧·덱·유물 변화 토스트
- 6단계 난이도, 난이도별 클리어/최고층/최근 시드 진행 기록, 커스텀 시드, 날짜/난이도별로 바뀌는 실제 규칙 조항 2개가 적용되는 일일 계약 런, 일일 계약 전용 승률/최고층/최근 조항 기록, 기록/요약 기반 시드 재도전, 기록 화면의 다음 목표 제안, 승리 후 다음 난이도 바로 도전, 구역별 적 풀, 난이도별 보스 후 구역 전환 회복, 자동 저장 상태 표시와 로컬 저장/이어하기 미리보기, 자동 저장 백업 복구와 손상 저장 알림, 저장된 런 보호를 위한 새 런 덮어쓰기 확인과 저장 삭제 확인, 원인 진단과 다음 런의 3가지 짧은 행동을 먼저 보여주는 런 요약, 전적 기록
- 런 진행 안내: 현재 막, 초반/중반/보스 직전 상태, 보스 이름과 특징, 보스까지 남은 층, 다음 선택지를 상단 띠와 맵 진행 레일에 표시하고, 보스가 가까워지면 체력/방어/마무리/정화·약화/카드 뽑기 점검을 함께 보여줍니다. 보스전에서는 보스 이름, 단계, 현재 의도, 전환 체력을 전투판 상단의 전용 막으로 강조합니다. 막 보스를 넘긴 뒤에는 회복량, 다음 구역, 다음 보스를 한 번에 보여주는 구역 클리어 안내가 맵에 한 번만 표시됩니다. 플레이 힌트는 현재 구간에서 먼저 볼 판단과 핵심 수치를 요약합니다.
- 게임 가이드: 첫 런에서 자주 막히는 경로 선택, 전투, 덱 정비, 보스 대비 판단과 여섯 가지 주력 전략을 시작 화면과 런 도중 언제든 확인
- 현재 덱 화면의 핵심 카드 분석, 평균 비용/강화/저주/덱 크기, 카드 종류 분포, 핵심 카드와 다음 보상에서 찾을 역할, 덱 정비 힌트
- 음량, 배경음 크기, 애니메이션 속도, 텍스트 크기, 고대비, 플레이 힌트 표시 설정
- 전투 중 선택 대상 표시, 가장 먼저 처리할 위험을 크게 보여주는 적 의도 패널, 좌측 하단의 에너지/더미 HUD, 카드별 피해/방어/뽑기/상태/자원 변화 미리보기, 그림 위 종류 배지와 첫 효과 강조, 숫자키 카드 사용/방향키 대상 전환/키보드 턴 종료, 작은 화면용 적/손패 터치 배치
- 층/전투 타입에 따라 바뀌는 전투 배경 아틀라스, 이벤트 전용 장소 아틀라스, 카드 내부 일러스트 전용 아틀라스, 적/보스 스프라이트 아틀라스, 드래그 시 카드에서 대상까지 이어지는 조준선, 카드 사용과 적 행동을 대상 방향에 맞춘 짧은 궤적·충격·숫자 피드백으로 보여주는 전투 FX, 처치 확인 뒤 보상으로 넘어가는 승리 연출, 카드/방어/피해/유물 발동 배너, 유물 상세/발동 시점/현재 덱에 맞는 이유를 보여주는 모달, 보상, 상점, 이벤트, 승리/패배용 브라우저 내장 효과음과 메뉴/탐사/전투/보스/승패용 음악 반복

## 조작

- 카드 클릭: 선택한 적에게 사용
- 카드 드래그: 마우스, 터치, 펜으로 원하는 적 위에 놓아 대상 지정과 사용을 동시에 수행
- `1-9`, `0`: 손패의 해당 위치 카드 사용
- 적 클릭: 대상 지정
- `←` / `→`: 전투 중 대상 전환
- `E` 또는 `Space`: 전투 중 턴 종료
- 전투 더미 카운트 클릭: 뽑기/손패/버림/소멸 더미의 카드, 평균 비용, 강화 수, 다음에 뽑을 카드 확인
- `턴 종료`: 적 행동 후 다음 턴 시작
- 맵 노드 클릭: 다음 장소 이동
- 시드 입력: 같은 시드로 동일한 맵 구조의 런 재도전
- `오늘의 계약`: 현재 날짜와 난이도에 고정된 시드와 계약 조항으로 일일 챌린지 시작
- 난이도 선택: 시작 화면의 캐릭터 패널에서 적용 후 시작 체력을 즉시 확인
- `D`: 런 도중 현재 덱 열기/닫기
- `Esc`: 덱/선택 모달 닫기 또는 설정/코덱스/기록에서 이전 화면으로 복귀
- 손패 카드 미리보기: 카드 하단과 툴팁에서 사용 후 피해, 방어, 뽑기, 상태, 자원 변화 확인
- 카드/상태 툴팁: 카드와 전투 상태 위에 마우스를 올려 키워드와 상세 효과 확인
- 유물 줄 클릭: 현재 보유 유물의 희귀도, 발동 시점, 효과, 현재 덱에 맞는 이유 확인
- `코덱스`: 런 중에도 카드, 유물, 적 행동, 키워드와 상태 효과 확인
- `가이드`: 런 중에도 초반 보상에서 주력 정하기, 생존 판단, 덱 정비, 보스 대비 요령, 주력 전략 빠른 보기 확인

## 세계관

플레이어는 `에코 다이버`가 되어 침수된 데이터 심해의 최심부로 내려갑니다. 카드는 기억 조각, 타이드 신호, 감염된 인덱스, 심해 장치로 구성되며, 유물은 탐사의 규칙을 뒤트는 장비입니다. 세 구역의 보스는 각기 다른 방식으로 현재 덱의 강점과 빈틈을 드러내며, 최종 심층에서는 `마지막 문 성가대`가 생존력과 마무리 수단을 함께 확인합니다.

## 주요 시스템

- 전투: 에너지, 뽑기/손패/버림/소멸 더미, 방어도, 취약, 약화, 균열, 바이러스, 표식, 전하, 집중, 반격, 도금, 보존, 소멸, 생성, 비용 변화가 실제 규칙으로 동작합니다. 전투 화면은 상단에 현재 위치와 적 의도, 중앙에 가장 먼저 처리할 위험, 좌측 하단에 에너지·더미 HUD, 하단에 손패와 대상 안내를 배치해 “무엇이 위험한가”와 “지금 어떤 카드를 쓸 수 있는가”가 같은 흐름으로 읽히게 합니다. 에너지가 부족한 카드는 비용 표시부터 잠겨 보이도록 처리해 카드 사용 가능 여부를 바로 알 수 있습니다. 턴 종료를 누르면 적 턴 배너와 함께 입력이 잠기고, 짧은 간격 뒤 실제 적 행동과 피해·방어·상태 결과가 해결된 다음 내 턴 배너로 손패가 다시 활성화됩니다. 턴 요약 패널은 현재 손패와 에너지를 기준으로 체력 손실을 막을 수 있는지, 어느 적을 처치할 수 있는지, 카드 뽑기와 에너지 여유가 어떤지 짧게 요약하고, 세부 판단을 열면 추천 사용 순서와 각 카드의 이유·효과를 함께 보여줍니다. 더미 점검 모달은 카드 목록, 평균 비용, 강화 수, 주요 종류, 다음에 뽑을 카드를 보여줘 핵심 카드를 다시 찾을 시점을 판단하게 합니다.
- 적 의도: 공격, 방어, 강화, 해로운 상태 부여, 충전 예고, 소환, 보스 전용 행동, 단계 전환 체력과 현재 단계가 의도 표시와 전투 로그에 나타납니다.
- 덱 빌딩: 시작 화면에서 캐릭터의 시작 덱 구성과 주력 전략을 확인한 뒤 런에 들어갑니다. 시작 덱은 공격/방어/전하/표식/정화 카드를 모두 포함합니다. 런 중 현재 덱 화면은 어떤 카드로 이길지, 평균 비용, 강화/저주 수, 덱 크기, 카드 종류 분포, 정비 힌트를 표시합니다. 전투 보상 3장 선택/받지 않기, 엘리트/보스/특수 전투 후 유물 3개 중 1개 선택, 상점 구매, 카드 제거, 휴식 강화, 이벤트 변환/복제/제거가 연결되어 있습니다. 보상 화면은 현재 덱과 유물에 맞는 이유, 선택 후 덱 크기와 평균 비용 변화, 중복 부담, 종류 비율, 이번 보상을 받지 않아도 좋은지, 유물별 현재 덱에 맞는 이유와 발동 시점을 함께 보여줍니다. 이벤트, 마켓, 세이프룸은 현재 막과 보스까지 남은 거리, 이번 선택의 추천 이유를 짧은 카드로 먼저 보여줘 선택지가 메뉴처럼 흩어지지 않게 합니다. 선택을 확정하면 체력, 크레딧, 덱 장수, 강화, 유물 획득 변화가 짧은 결과 토스트로 떠서 방금 결정한 의미를 바로 확인할 수 있습니다.
- 반복 플레이: 절차 생성 맵, 직접 입력한 시드 재도전, 요약 화면의 즉시 시드 재도전과 승리 후 다음 난이도 도전, 날짜/난이도 기반 오늘의 계약, 오늘의 계약 전용 기록, 구역별 적 풀, 현재 막 진행도와 보스까지 남은 층을 보여주는 런 안내, 완료/선택 가능/미확정 경로를 구분하는 맵 연결선, 각 경로의 위험/보상/다음 분기와 적 후보/보스 특징/상점·휴식 자원 안내, 난이도별 보스 후 회복과 구역 완료 안내, 최종 승리 후 코어 회수 장면, 이벤트 중복 방지, 난이도 6단계, 자동 저장과 복구, 저장된 런 보호 흐름, 다음 런 행동이 먼저 보이는 런 요약, 기록 화면의 다음 목표 제안, 최고 기록, 최근 런 기록을 제공합니다.
- 접근성: 음량, 배경음 크기, 애니메이션 속도, 텍스트 크기, 고대비 모드를 설정할 수 있고 카드 종류/희귀도/상태 효과는 색상뿐 아니라 텍스트와 카드/상태 툴팁으로 함께 전달됩니다. 유물은 아이콘만이 아니라 상세 모달과 툴팁에서 이름, 희귀도, 발동 시점, 효과, 현재 덱에 맞는 이유를 텍스트로 제공합니다. 전투 중에는 선택 대상, 적 의도, 예고 피해, 해로운 상태, 턴 요약, 보스 대비 점검이 텍스트로 요약되어 터치 환경에서도 카드 대상과 생존 판단이 분명하게 유지됩니다. 손패와 턴 종료 버튼에는 시각적 키 배지와 `aria-keyshortcuts`가 붙어 키보드만으로 카드 사용, 대상 전환, 턴 종료가 가능합니다.
- 아트 방향: 전투 배경은 `arena-backdrops.png`의 6개 배경 셀을 층, 전투 타입, 현재 적 조합에 따라 바꿔 쓰고, `arena-props.png`의 전경 오브젝트 셀을 추가로 얹어 기록실 케이블, 관측실 링, 산호 가지, 서버 기둥, 심해 결정, 마지막 문 실루엣이 전투마다 다르게 보이게 합니다. 같은 셀이라도 조명 방향, 안개 위치, 카메라 줌, 전경 레이어가 달라져 매 전투가 다른 장소처럼 보이도록 구성했습니다. 카드 내부 이미지는 `card-illustrations.png`의 71개 일러스트 셀을 카드 프레임 비율에 맞춘 전용 이미지 레이어로 올리고, 스타터 카드는 덧그린 문양 없이 페인팅 원본이 먼저 보이게 했습니다. 다른 카드도 과한 선형 심볼 대신 낮은 강도의 조명과 색 보정만 더해 카드 그림 자체가 중심이 되도록 구성했습니다. 보상과 상점 화면의 카드도 손패와 같은 시각 문법을 사용해 이름, 그림, 종류, 효과가 같은 순서로 읽히게 했습니다. 플레이어, 일반 적, 엘리트, 보스는 `combatants/`의 투명 PNG 스프라이트로 전투판 위에 서 있고, 공격 의도와 상태 아이콘은 `sprite-atlas.png`를 분리해 사용합니다. `enemy-portraits.png`와 `combat-sprites.png`는 보조 아틀라스와 안전한 폴백으로 남겨 두었습니다. 이벤트 화면은 `event-backdrops.png`의 6개 장소 셀을 사건 종류와 위험도에 맞춰 바꾸고, 선택지의 위험도와 추천 선택에 맞춰 조명만 추가로 얹습니다. 보스전은 보스별 배경과 2단계 진입 시 화면 펄스 연출을 사용해 일반 전투와 구분됩니다. 모든 시각 요소는 프로젝트 전용으로 제작되어 외부 저작권 IP를 사용하지 않습니다.
- 코덱스와 가이드: 모든 카드의 비용/종류/희귀도/강화 설명, 유물 발동 시점, 적 의도 목록, 키워드/상태 효과 정의를 코덱스에서 확인할 수 있고, 가이드 화면은 첫 런의 경로 선택, 전투 읽기, 덱 정비, 보스 대비 판단과 전하 모아 한 번에 쓰기, 표식 남기고 연달아 공격, 바이러스와 약화로 버티기, 반격으로 되받기, 필요한 카드 다시 찾기, 대가를 내고 더 행동하기를 짧게 정리합니다.
- 사운드와 화면 반응: 지원 브라우저에서는 내장 오디오 합성으로 카드 사용, 피해, 방어, 보상, 상점, 이벤트, 보스 2단계 전환, 승리/패배 효과음과 메뉴/탐사/전투/보스/보상/승패 상태별 음악 반복을 만듭니다. 보스 2단계 전환 시에는 전용 효과음, 화면 충격막, 보스별 2단계 음악으로 전환됩니다. 카드와 적 행동은 대상 방향에 맞춘 짧은 궤적, 타격 지점 충격, 피해/방어/상태 숫자, 대상 강조로 표시됩니다. 여러 적이 한 번에 행동하면 효과를 여러 번 띄우지 않고 `×2` 같은 묶음 표시와 대표 스프라이트로 한 번에 보여줘 누가 무엇을 했는지 읽히게 했습니다. 카드 호버는 카드 상세에 집중하고, 드래그 중에는 조준선과 대상 미리보기를 보여줘 정보가 한꺼번에 겹치지 않게 했습니다. 적을 처치하면 바로 보상만 띄우지 않고 마무리 카드, 처치한 적, 보상 예고를 순서대로 보여주는 확인 연출을 거칩니다. 광역 카드는 모든 적에게 대상 표시를 띄워 어느 적에게 영향을 주는지 바로 알 수 있습니다. 전투, 보상, 맵, 상점, 휴식, 이벤트로 넘어갈 때는 짧은 전환 배너가 현재 목표와 핵심 수치를 보여줍니다. 유물은 발동 시 아이콘 강조와 배너로 즉시 표시되며, 지원하지 않는 환경에서는 무음으로 안전하게 동작합니다.
